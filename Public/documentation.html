What is an API?


This is my API (Application Programming Interface) for this project. An API help connect the users application with backend systems.

Table

Request URL HTTP Method Response
Allow new users to register "/users" POST Will add new user to JSON object.
Allow users to add a movie to their list of favorites "/users/:_id/:movieTitle" POST Returns a text confirming which movie was added successfully.
Return a list of all the movies to the user '/movies' GET Returns a JSON list of movies.
Return data about a single movie by title to the user '/movies/:title' GET Returns a JSON data about a specific movie.
Return data about a director by name "/movies/directors/:directorName" GET Returns a JSON data about the director.
Return information about a genre by genre name. "/movies/genre/:genreName" GET Returns a JSON data with a description of that genre.
Allow users to update their user information "/users/:Username" PUT Returns the data fields that was updated by the user.
Allow users to remove a movie from their list of favorites "/users/:_id/:movieTitle" DELETE Returns a text confirming which movie was removed successfully.
Allow existing user to deregister /users/:Username DELETE Return a text confirming that the user was deleted.
